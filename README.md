# Laboratorio 4 - Scripting en Bash

## Descripción general
Este repositorio contiene la resolución del Laboratorio 4 de Sistemas Operativos. El trabajo fue realizado en Bash sobre una máquina virtual Ubuntu 24.04 en Microsoft Azure.

En este laboratorio se desarrollaron scripts para mostrar información del sistema, listar procesos, automatizar la copia de archivos y reunir todo en un menú interactivo.

## Funcionalidades principales
- Mostrar información del sistema
- Listar procesos ordenados por uso de CPU
- Copiar archivos modificados en las últimas 24 horas
- Ejecutar todos los retos desde un menú interactivo

## Instrucciones de ejecución
Primero se deben dar permisos de ejecución a los scripts:

```bash
chmod +x reto1_info_sistema.sh
chmod +x reto2_procesos.sh
chmod +x reto3_automatizacion.sh
chmod +x lab4_menu.sh
```

## Luego se pueden ejecutar de la siguiente manera:
```bash
./reto1_info_sistema.sh
./reto2_procesos.sh
./reto3_automatizacion.sh
./lab4_menu.sh
```

## Entorno utilizado
- Ubuntu Server 24.04 LTS
- Microsoft Azure
- Conexión SSH desde Windows
- Lenguaje de scripting: Bash
